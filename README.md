## Install and run locally

#### Clone repo

```

$ https://github.com/JacobWylie/grover-chatbot.git

```

####CD into directory

```

$ cd grover-chatbot 

```

#### Install modules

```

$ npm install

```

#### Run Node server on port 8085

```

$ node src/app.js

```

#### Start webpack server (if browser does not automatically open, navaigate to http://localhost:3000/)

```

$ npm start

```
